# SweetDream Portfolio

This repository showcases the architecture, implementation, and technology stack used for the SweetDream system — a cross-platform monitoring and control solution consisting of:

- Android Application (Remote Monitoring Client)
- Backend API Server (Node.js + SQLite)
- Desktop Controller Application
- Event System & File Upload Pipeline
- Command System (Camera, Audio, Storage, Calls, Notifications)
- Firebase Cloud Messaging (FCM)

Full documentation is available in the **/docs** folder.
